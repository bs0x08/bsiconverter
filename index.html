Image Converter Tool

bsiConverter is a portable Windows application for converting images between popular formats (JPEG, PNG, BMP, GIF, TIFF, WEBP). It features a simple drag-and-drop interface, batch and single file support, and tracks conversion history in real time. The program is multilingual (English and Russian) and requires no installation.

Key Features:

  • Convert images to JPEG, PNG, BMP, GIF, TIFF, WEBP
  • Drag-and-drop support + classic file dialog
  • Tabbed interface (Converter, History, Help)
  • Adjustable JPEG quality (Low / Medium / High / Maximum)
  • Automatic color mode handling (e.g., RGBA → RGB for JPEG)
  • Real-time metadata display (name, size, resolution, format)
  • Session history log with file size, format and date
  • Quick "Open Folder" button
  • Built-in help & localized interface (English, Russian)
  • Portable — no installation required

System Requirements:

  • OS: Windows 7/8/10/11 (64-bit)
  • RAM: 512 MB (1 GB recommended)
  • Disk Space: 50–100 MB
  • Works on Linux/macOS via Wine (not guaranteed)
