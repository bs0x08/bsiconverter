Project Description

bsiConverter is a modern Windows application for converting images between popular formats (JPEG, PNG, BMP, GIF, TIFF, WEBP). It features drag-and-drop support, a clean tabbed interface, conversion history tracking, and multilingual support (English & Russian). The program is portable and requires no installation.

Technical Details

  • Embedded Resources: Base64 icons and images (no external resource files)
  • Platform: Windows 7/8/10/11 (official), works on Linux/macOS via Wine (not guaranteed)
  • Architecture: 64-bit (x86_64)
  • Data Storage: Session-based (conversion history is cleared after restart)

Main Features

  • Convert images to JPEG, PNG, BMP, GIF, TIFF, WEBP
  • Adjustable JPEG quality (Low / Medium / High / Maximum)
  • Automatic color mode handling (e.g., RGBA → RGB for JPEG)
  • Tabbed interface (Converter, History, Help)
  • Drag-and-drop & classic file dialog
  • Real-time metadata display (name, size, resolution, format)
  • Conversion history log (date, format, size, quality)
  • Quick “Open Folder” button
  • Built-in help section & “About” dialog with embedded graphics
  • Multilingual interface: English and Russian

Limitations

  • No direct support for RAW or HEIF/HEIC formats
  • No command-line interface (GUI only)
  • Conversion history is not saved permanently (session only)
  • Linux/macOS support via Wine may have limited functionality
